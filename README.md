# quick-links-importer
快速批量导入链接到WordPress链接管理器，支持JSON格式导入，包含名称、网址、简介、Logo

介绍文章：[WordPress批量导入链接，助力快速构建导航](https://wxsnote.cn/7261.html)
信息抓取工具也在上文，插件请在发行版下载，不要直接下载仓库，仓库是最新内容，但并不稳定，不推荐下载使用，后续可能上架wordpress官方插件目录
